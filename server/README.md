# server/ — MCP server (WS-D)

MCP server exposing FDTDMEX to LLM orchestration: introspect available types/params, build/edit/ validate a declarative setup, run a simulation, and fetch results. Built on the pydantic config schema in `fdtdmex.io`.

Install with the extra: `uv sync --extra mcp`.

See [../docs/mcp-and-ui.md](../docs/mcp-and-ui.md). Status: stub.
